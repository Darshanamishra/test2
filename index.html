Project Details

This project involved creating an interactive sales dashboard for Speedle Bikes using Power BI, leveraging a robust data pipeline and advanced analytics.

Workflow:


    1. Data Gathering: Extracted data from a robust MySQL database using comprehensive SQL queries to ensure relevant and accurate data for analysis.

    2. Data Cleaning & Exploration: Performed in-depth data cleaning and exploratory data analysis using MySQL DBMS, identifying key data patterns and ensuring data quality for further analysis.

    3. Data Connectivity: Integrated MySQL with Power BI through ODBC connections, enabling seamless data flow and real-time updates for dynamic dashboards.

    4. Data Transformation: Utilized Power Query for efficient data transformation, including data wrangling, cleaning, and reshaping to optimize the dataset for analysis.

    5. DAX Measures & Visualizations: Developed and implemented advanced DAX measures to create insightful and interactive visualizations, such as KPI matrices, trend analysis, and Dashboard Reporting for performance tracking and decision-making support.

Project Details

In this project, I analyzed auto insurance claims data using Excel pivot tables, Power Query, and various data visualization techniques. The goal of the project was to help insurance companies better understand risk factors, claim trends, and customer behavior, leading to more informed decision-making processes.

Key deliverables included a dynamic, interactive dashboard that provided insights on claims across different time periods and geographic regions. The dashboard allowed users to filter data based on multiple criteria such as claim type, policy period, and customer demographics, enabling more tailored strategies for risk mitigation and customer service optimization.

Throughout this project, I have demonstrated my deep understanding and practical knowledge in the following areas::


    1. Data Transformation Techniques

    2. Pivot Table

    3. Power Query

    4. Exploratory Data Analysis

    5. Conditional Formatting

    6. Risk Factors Metrics

    7. Interactive Dashboard

The insights derived from the analysis helped the insurance company improve operational efficiency, reduce fraud, and maximize profitability by identifying high-risk areas and optimizing resource allocation.
